# Task: Criar Anuncio de Resposta Direta

## Contexto
Criar um anuncio de resposta direta completo usando a metodologia Amanda Khayat.

## Prerequisitos
- Definir o produto/servico
- Definir o publico-alvo (o peixe)
- Ter acesso a pesquisa de mercado ou concorrentes

## Pipeline de Execucao

### 1. Pesquisa do Consumidor
- Pensar como o peixe, nao como o pescador
- Mapear estado atual vs estado desejado do consumidor
- Pesquisar linguagem real (Facebook/YouTube, NAO TikTok)
- Usar perfil falso de consumidor pra receber conteudo organico

### 2. Briefing Pre-Escrita (4 Componentes)
- **Big Idea:** Qual eh a UNICA ideia que redefine o produto?
- **Avatar:** Quem vai falar no anuncio? Qual idade, genero, contexto?
- **Justificativa:** Por que ESTA ideia e ESTE avatar?
- **Aprovacao:** Aprovar brief ANTES de escrever

### 3. Producao (Metodo 25)
- Definir 5 tipos de gancho (organico, validado, estrutura invisivel, do zero, outro nicho)
- Definir 5 tipos de corpo
- Cruzar pra gerar 25 combinacoes unicas
- Max 8 anuncios por dia — qualidade sobre quantidade

### 4. Otimizacao (7 Alavancas)
- Verificar Formato/Avatar
- Verificar Hook Visual + Escrito
- Inserir Frase de Pouso pos-gancho
- Checar Congruencia
- Expandir Contexto do Avatar
- Inserir Gatilhos de Dopamina (sem conectores)
- Definir Fechamento (Push-Pull ou Autoridade Humanizada)

### 5. Validacao
- Cada frase classificada como Promessa, Prova ou Curiosidade
- Frases de um elemento so: reescrever pra combinar dois ou tres
- Verificar que o anuncio NAO parece anuncio
- Conferir congruencia avatar-conteudo-produto

## Output Esperado
- Briefing aprovado
- Anuncio completo (gancho + frase de pouso + corpo + gatilhos de dopamina + fechamento)
- Classificacao Promessa/Prova/Curiosidade de cada frase

## Regras
- IA pode ajudar a PENSAR, nunca a ESCREVER
- Sem Big Idea = sem anuncio
- Sem briefing = sem anuncio
