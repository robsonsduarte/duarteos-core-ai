# Task: Desconstruir Anuncio (Estrutura Invisivel + Mecanismos)

## Contexto
Desconstrucao completa de um anuncio — proprio ou de concorrente — para extrair a estrutura invisivel, mapear mecanismos psicologicos e gerar insights para novos anuncios.

## Input
- Anuncio completo (texto ou transcricao de video)
- Nicho/produto
- Metricas de performance se disponiveis

## Pipeline de Execucao

### 1. Mapeamento de Elementos
Isolar cada elemento do anuncio:
- **Hook:** Qual gancho? Visual ou verbal? Quanto tempo?
- **Frase de Pouso:** Existe? Qual? Abre loop?
- **Big Idea:** Qual eh? De qual dominio veio?
- **Avatar:** Quem fala? Congruente?
- **Corpo:** Qual estrutura? Testemunho? Explicacao? Lista?
- **Gatilhos de Dopamina:** Tem? Onde? Com ou sem conectores?
- **Fechamento:** Push-Pull? Autoridade Humanizada? CTA direto?

### 2. Classificacao PPC (Promessa/Prova/Curiosidade)
- Classificar CADA frase como P, P ou C
- Identificar frases multi-elemento
- Calcular balanco geral do anuncio

### 3. Analise dos 7 Alavancas
Avaliar cada alavanca (1-10):
1. Formato/Avatar: ___/10
2. Hook Visual + Escrito: ___/10
3. Frase de Pouso: ___/10
4. Congruencia: ___/10
5. Contexto do Avatar: ___/10
6. Gatilho de Dopamina: ___/10
7. Fechamento: ___/10

### 4. Extracoes de POR QUE
Para cada elemento forte:
- POR QUE funciona? (mecanismo psicologico)
- Como criar algo NOVO que gere o mesmo efeito?
- Qual principio classico esta por tras? (Halbert, Schwartz, Hopkins)

### 5. Mapeamento de Estrutura Invisivel
- Remover todo conteudo de superficie
- Descrever a estrutura PURA: [tipo gancho] -> [tipo Frase de Pouso] -> [tipo corpo] -> [tipo gatilho] -> [tipo fechamento]
- Essa estrutura pode ser usada pra criar anuncio novo?

### 6. Analise de Erros (se anuncio nao performou)
- "A gente analisa muito o que deu bom, mas por que que nao analisa o que nao deu bom?"
- Onde o anuncio perdeu a pessoa?
- Qual alavanca estava fraca?
- Avatar era incongruente?
- Faltou Big Idea?

## Output Esperado
- Mapa estrutural completo
- Score das 7 Alavancas
- Classificacao PPC frase-a-frase
- Estrutura Invisivel extraida (reutilizavel)
- Analise POR QUE funciona (ou nao funciona)
- 3+ insights acionaveis pra novos anuncios

## Regras
- "O hack nao eh replicar o que da certo. Eh entender POR QUE da certo."
- Nao julgar por gosto pessoal — se escala, funciona
- Analisar tanto vencedores quanto perdedores
