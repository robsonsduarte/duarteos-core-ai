# Task: Otimizar Gancho (Hook)

## Contexto
Otimizar ou criar novos ganchos para anuncios usando as tecnicas Amanda Khayat. O gancho decide se a pessoa para ou passa. Eh o elemento mais critico do anuncio.

## Input
- Ganchos atuais (se existentes) com metricas
- Produto e Big Idea
- Avatar definido
- Formato do anuncio (video UGC, imagem, carrossel)

## Pipeline de Execucao

### 1. Avaliar Gancho Atual
- Prende nos primeiros 3 segundos?
- Tem hook visual ANTES do hook verbal?
- O gancho eh congruente com o avatar?
- Gera curiosidade ou so informa?

### 2. Frase de Pouso
- O gancho atual TEM Frase de Pouso?
- A Frase de Pouso eh tao forte quanto o gancho?
- Abre um loop de curiosidade/controversia?
- "A frase de pouso que vem pos gancho tem que abrir um loop, tem que ser muito curiosa, tem que gerar controversia"

### 3. Gerar Variacoes (5 Tipos)
1. **Organico:** Adaptar gancho de conteudo organico que ja viralizou
2. **Validado do nicho:** Modelar gancho que funciona no mesmo nicho
3. **Estrutura Invisivel:** Extrair estrutura de gancho vencedor, reconstruir com conteudo novo
4. **Do zero:** Criar original baseado na Big Idea e no desejo do consumidor
5. **Cross-nicho:** Adaptar gancho de nicho diferente

### 4. Entender POR QUE
- Para cada gancho: POR QUE prende atencao?
- Qual mecanismo psicologico? (curiosidade, choque, identificacao, prova social)
- "Voces nao tentam entender o porque daquele padrao"

### 5. Teste de Congruencia
- O gancho eh congruente com o corpo do anuncio?
- O gancho eh congruente com o produto?
- O gancho eh congruente com o avatar?
- "Alguem usando avatar de velha de 90 anos falando giria de adolescente — nao faz sentido"

### 6. Adicionar Frase de Pouso a Cada Variacao
- Criar Frase de Pouso pra cada gancho novo
- Cada Frase de Pouso deve abrir loop diferente
- Testar combinacao gancho + Frase de Pouso

## Output Esperado
- 5+ ganchos novos com Frase de Pouso
- Classificacao POR QUE cada um funciona
- Checklist de congruencia
- Recomendacao de teste (ordem de prioridade)

## Regras
- Gancho visual vem ANTES do gancho verbal
- Frase de Pouso eh OBRIGATORIA
- Nao copiar gancho — entender POR QUE funciona e criar novo
- "Todos os ganchos que eu mais validei foram: analisei a estrutura invisivel desse gancho que ta validado e criei uma nova"
