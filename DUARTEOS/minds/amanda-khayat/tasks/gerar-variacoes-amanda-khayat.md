# Task: Gerar Variacoes de Anuncio (Metodo 25)

## Contexto
Gerar variacoes sistematicas de um anuncio existente usando o Metodo 25 — 5 ganchos x 5 corpos.

## Input
- Anuncio original (o controle ou ponto de partida)
- Produto/servico
- Publico-alvo definido
- Big Idea definida

## Pipeline de Execucao

### 1. Definir 5 Tipos de Gancho
1. **Organico:** Buscar no conteudo organico que ja validou (viral, engajamento alto)
2. **Validado do nicho:** Modelar ganchos que ja funcionam no nicho do produto
3. **Estrutura Invisivel:** Extrair estrutura de gancho vencedor e reconstruir com conteudo novo
4. **Do zero:** Criar gancho 100% original baseado na Big Idea
5. **De outro nicho:** Adaptar gancho vencedor de nicho diferente (cross-pollinacao)

### 2. Definir 5 Tipos de Corpo
- Variar formato, tom, estrutura narrativa
- Cada corpo eh uma abordagem diferente para contar a historia da Big Idea
- Exemplos: testemunho pessoal, explicacao tecnica, historia dramatica, lista de beneficios, comparativo

### 3. Cruzar e Produzir
- Cruzar os 5 ganchos x 5 corpos = 25 combinacoes
- Escrever max 8 por dia
- Cada combinacao eh unica — nao eh repetir com sinonimos

### 4. Aplicar 7 Alavancas a Cada Uma
- Verificar Frase de Pouso em cada variacao
- Inserir Gatilhos de Dopamina
- Checar congruencia avatar-conteudo
- Definir fechamento (Push-Pull ou Autoridade Humanizada)

### 5. Classificar e Priorizar
- Usar Promessa/Prova/Curiosidade pra avaliar cada variacao
- Priorizar as que combinam mais elementos
- Marcar as top 5-8 para teste primeiro

## Output Esperado
- 25 combinacoes unicas (gancho + corpo)
- Top 5-8 priorizadas para teste
- Classificacao PPC de cada uma

## Regras
- Max 8 por dia — "Se eu escrever 16, vou fazer 8 novos que sao uma bosta. E os 8 originais tambem ficam bosta."
- Cada combinacao eh UNICA, nao sinonimos
- Manter Big Idea como centro gravitacional de todas as variacoes
