# Task: Escrever Variantes de Headline (Joanna Wiebe Method)

## Objetivo
Gerar 10-20 variantes de headline baseadas em VOC data, usando os frameworks de Joanna Wiebe — Review Mining, Already/Still, Even/If, Specificity Zoom, e Contrarian Challenge. "The best headlines come verbatim from customer reviews."

## Processo

### Fase 1: VOC Data (Pre-requisito)
**Se nao tem VOC data, PARE e faca review mining primeiro.**
- Precisa de: pain language, desire language, trigger moments, objecoes
- Fonte: Amazon reviews, Reddit, Facebook groups, surveys, interviews

### Fase 2: Formulas de Headline

**Formula 1: VOC Verbatim**
- Tome a linha mais poderosa de uma review e use como headline
- "If you think you need rehab, you do" — 26% more leads
- Teste verbatim contra versao editada

**Formula 2: Already/Still**
- "You've already [qualificacao]. But you're still [gap]."
- Valida o prospecto E cria tensao construtiva

**Formula 3: Even/If**
- "[Beneficio] — even if [objecao mais comum]"
- Pre-handles a objecao inline

**Formula 4: Specificity Zoom**
- Substitua cada palavra generica por detalhe concreto do VOC
- "Save time" → "Get your Tuesday afternoons back"
- Teste: "O prospecto consegue VER isso na mente?"

**Formula 5: Contrarian Challenge**
- "Since when did we [ortodoxia da industria]?"
- Desafia crenca comum com dado surpreendente

**Formula 6: Pain Language Specific**
- Use a linguagem EXATA de dor do cliente
- "My armpits are ALWAYS wet" (108% lift)
- Quanto mais especifica e vivida, melhor

**Formula 7: Desire Language Specific**
- Use a linguagem de resultado desejado do cliente
- Focada no OUTCOME, nao no processo

**Formula 8: Question (Rhetorical)**
- Pergunte o que o prospecto ja esta perguntando silenciosamente
- NAO faca perguntas genericas — especificas ao problema

### Fase 3: Geracao
Para cada formula, gere 2-3 variantes. Total: 16-24 headlines.

### Fase 4: Priorizacao
Rankeie por:
1. **Especificidade** — pode ser visualizada?
2. **VOC proximity** — quao perto esta da linguagem real?
3. **Emotional charge** — provoca emocao?
4. **One big idea** — comunica uma unica ideia clara?

### Fase 5: Teste
- Selecione top 3-5 para teste A/B
- Teste em pares (controle vs variante)
- "Only data determines what converts"

## Output Esperado
- 16-24 variantes de headline organizadas por formula
- Top 5 rankeadas com justificativa
- Plano de teste A/B (pares de teste)
- VOC source citada para cada headline

## Checklist de Qualidade
- [ ] VOC data foi fonte primaria? (nao imaginacao do copywriter)
- [ ] Cada headline pode ser visualizada pelo prospecto?
- [ ] Already/Still e Even/If formulas incluidas?
- [ ] Pelo menos 3 headlines sao VOC verbatim?
- [ ] Cada headline comunica UMA big idea?
- [ ] Headlines rankeadas por especificidade?
- [ ] Plano de teste definido?
- [ ] Nenhuma headline generica ("save time and money")?
- [ ] Linguagem do cliente preservada?
